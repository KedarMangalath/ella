# Ella Project Instructions

Build Ella as a serious TypeScript CLI coding agent.

Priorities:

- Keep core model/provider logic decoupled from terminal UI.
- Prefer small, testable modules.
- Keep tool execution permissioned and auditable.
- Avoid hard-coding one model provider.
- Keep old `Agentic/` projects as reference only.

Near-term roadmap:

1. Strong CLI MVP.
2. Better streaming output.
3. Native MCP client.
4. Session compaction and project memory.
5. code-review-graph bridge.
6. Subagents and background tasks.
